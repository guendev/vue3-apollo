# AI Skills

This repository provides an agent skill named **`vue3-apollo`**.

## About

`vue3-apollo` is a reusable instruction package for coding agents.

It helps agents work with:

- Vue 3
- Nuxt 4
- composables: `useQuery`, `useLazyQuery`, `useMutation`, `useSubscription`, `useFragment`, `useAsyncQuery`
- migration, caching, SSR, tracking, and troubleshooting

## Install

Install from this repository:

```bash
pnpx skills add guendev/vue3-apollo --skill vue3-apollo
```
